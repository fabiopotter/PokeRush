# 📋 Checklist Editorial - PokeRush

## ✅ Pré-Publicação

### Conteúdo
- [ ] **Título** inclui palavra-chave principal no início?
- [ ] **Resumo** é curto e atrativo (2-3 linhas)?
- [ ] **Estrutura** usa títulos, subtítulos e listas para leitura rápida?
- [ ] **Linguagem** é direta, em português brasileiro, sem enrolação?
- [ ] **Utilidade** resolve dúvida específica do usuário rapidamente?
- [ ] **Originalidade** não é tradução literal de sites estrangeiros?
- [ ] **Concisão** remove palavras desnecessárias e repetições?

### SEO
- [ ] **Palavra-chave** aparece no título e primeira frase?
- [ ] **Subtítulos** (H2, H3) organizam o conteúdo logicamente?
- [ ] **Variações** naturais da palavra-chave estão incluídas?
- [ ] **FAQ** foi adicionado quando relevante?
- [ ] **Links internos** conectam a outros conteúdos do site?

### Qualidade
- [ ] **Termos técnicos** são explicados quando usados?
- [ ] **Informações** estão atualizadas e corretas?
- [ ] **Exemplos práticos** estão incluídos?
- [ ] **Conclusão** resume pontos principais?
- [ ] **Mobile-first** otimizado para leitura em dispositivos móveis?

## 🎨 Formatação

### Estrutura
- [ ] **Título principal** (H1) é claro e atrativo?
- [ ] **Subtítulos** (H2, H3) quebram conteúdo em seções?
- [ ] **Listas** são usadas para passos sequenciais ou itens?
- [ ] **Destaques** usam negrito para informações importantes?
- [ ] **Parágrafos** são curtos (3-5 linhas) para leitura fácil?

### Mídia
- [ ] **Imagem de capa** está presente e relevante?
- [ ] **Alt text** é descritivo para acessibilidade?
- [ ] **Screenshots** do jogo são usados quando possível?
- [ ] **Tabelas** são claras e bem formatadas?

## 📖 Formatos Específicos

### Para Guias
- [ ] **Introdução** vai direto ao ponto (sem "Olá, treinadores!")?
- [ ] **Seções** são lógicas e bem organizadas?
- [ ] **Passos** usam listas numeradas quando sequenciais?
- [ ] **Dicas** são práticas e acionáveis?
- [ ] **Conclusão** resume pontos principais?
- [ ] **FAQ** responde perguntas reais dos usuários?

### Para Pokédex
- [ ] **Descrição** é curta e objetiva (1-2 frases)?
- [ ] **Informações** incluem tipo, fraquezas e resistências?
- [ ] **Localização** especifica onde encontrar por jogo?
- [ ] **Uso prático** explica aplicação competitiva e casual?
- [ ] **Evoluções** mostram linha completa com requisitos?

### Para Notícias
- [ ] **Título** é impactante e inclui palavra-chave?
- [ ] **Resumo** explica o que aconteceu em 2-3 linhas?
- [ ] **Detalhes** fornecem informações principais?
- [ ] **Impacto** explica como afeta os jogadores?
- [ ] **Próximos passos** indicam o que esperar?

## 🔍 Revisão Final

### Linguagem
- [ ] **Tom** é amigável e direto (como conversa com amigo)?
- [ ] **Vocabulário** usa palavras comuns do dia a dia?
- [ ] **Explicações** são simples para conceitos complexos?
- [ ] **Evitou** jargões desnecessários?

### Engajamento
- [ ] **Valor** o conteúdo agrega valor real ao usuário?
- [ ] **Motivação** encoraja sem ser excessivo?
- [ ] **Ação** inclui próximos passos claros?
- [ ] **Contexto** considera nível do jogador (iniciante/avançado)?

### SEO Final
- [ ] **Meta title** otimizado (50-60 caracteres)?
- [ ] **Meta description** atrativa (150-160 caracteres)?
- [ ] **URL** é limpa e inclui palavra-chave?
- [ ] **Estrutura** facilita navegação e indexação?

---

## 🚨 Sinal Vermelho
Se qualquer item abaixo for verdadeiro, **REVISAR** o conteúdo:

- ❌ Introdução genérica ("Olá, treinadores!")
- ❌ Texto longo sem valor prático
- ❌ Tradução literal de sites estrangeiros
- ❌ Repetições desnecessárias
- ❌ Profundidade excessiva em tópicos básicos
- ❌ Linguagem formal demais ou muito informal
- ❌ Falta de estrutura escaneável
- ❌ Não resolve dúvida específica do usuário

## 📊 Pós-Publicação
Após publicar, verificar:
- [ ] Tempo médio de leitura (meta: 2-3 minutos)
- [ ] Taxa de conclusão (meta: 80%+)
- [ ] Posicionamento SEO para palavras-chave
- [ ] Engajamento social e comentários
